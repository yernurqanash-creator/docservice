DocService API

Upload: multipart/form-data, field name: file

Endpoints

POST /convert-smart-json   (PDF/DOCX/PNG/JPG/ZIP) → JSON chunks
POST /convert-smart-md     (PDF/DOCX/PNG/JPG/ZIP) → Markdown text

curl examples

curl -X POST "http://localhost:8091/convert-smart-json" ^
  -H "Content-Type: multipart/form-data" ^
  -F "file=@kundelik.pdf"
curl -X POST "http://localhost:8091/convert-smart-md" ^
  -H "Content-Type: multipart/form-data" ^
  -F "file=@kundelik.docx"

Response example (JSON)

{
  "format": "markdown",
  "meta": { "filename": "...", "contentType": "...", "sizeBytes": 123 },
  "chunksCount": 8,
  "chunks": [
    { "index": 0, "chars": 1400, "text": "..." }
  ]
}
